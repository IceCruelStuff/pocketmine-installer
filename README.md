## Development
#### Prerequisites
* [NSIS](https://nsis.sourceforge.io/Main_Page)
* [Inetc Plug-in](https://nsis.sourceforge.io/Inetc_plug-in)
* [ZipDLL Plug-in](https://nsis.sourceforge.io/ZipDLL_plug-in)

#### Compiling
```sh
makensis "PocketMine-MP.nsi"
```
